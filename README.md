# 🎙️ QRally Dictate
**A high-performance, offline-ready dictation tool for Rally Co-Drivers.**

QRally Dictate allows you to record stage notes, listen back to them, and share them instantly via WhatsApp—even when signal is low.

## 🚀 Quick Start
1. **Open the App:** Visit [https://qrally.github.io/QRally-Dictate/](https://qrally.github.io/QRally-Dictate/)
2. **Install to Home Screen:** - **iOS:** Tap 'Share' -> 'Add to Home Screen'.
   - **Android:** Tap 'Options' -> 'Install App'.
3. **Dictate:** Hold the big button, speak your note, and release.

## 🛠 Features
- **PTT (Push-To-Talk):** Mimics radio hardware for intuitive use under pressure.
- **Offline Engine:** Works in remote locations thanks to Service Worker technology.
- **Rally Ready:** High-contrast "Rally Yellow" theme and haptic vibration feedback.
- **Data Safety:** All notes are saved to your device's local storage.
